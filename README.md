# DeliverMyFood APP

MERN-based delivery management portal for the **DeliverMyFood** ecosystem, with real-time order tracking, dashboard stats, and seamless database integration.

## Getting Started

### Prerequisites
- Node.js (version 18 or above)
- MongoDB running locally on port `27017`

### 1. Run the Backend Server
```bash
cd server
npm install
node user-service.js
```
The server will run on port `5010`.

### 2. Run the Frontend
```bash
npm install
npm run dev
```
The frontend will run on `http://localhost:5175`.
